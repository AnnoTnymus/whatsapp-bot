# Presentación Comercial v2

## Archivos

- **index.html** — Presentación interactiva (slides con navegación)
- **imprimir.html** — Versión optimizada para PDF

## Cómo usar

### Presentación interactiva
Abrir `index.html` en navegador y navegar con:
- **Flechas** o **Espacio** — siguiente slide
- **Click** en botones de navegación

### Generar PDF
1. Abrir `index.html` o `imprimir.html`
2. Ctrl+P / Cmd+P
3. Guardar como PDF (configurar márgenes mínimos)
